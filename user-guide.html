Keeper User Guide

Keeper is an inbox-first place for quick notes, links, snippets, screenshots, and loose thoughts. Capture quickly, organize when useful, then export or archive notes when they have served their purpose.

Fast capture Inbox first Tags and smart views Export and burn

Overview

The main screen is built around a note stream. The sidebar switches between views, the top bar handles selection and bulk tools, and the note grid shows your current set of notes in a card layout.

Keeper works well as a temporary holding area: toss things into the Inbox, find them again with search or tags, archive notes that are handled, and export selected material when it needs to move elsewhere.

Quick Add

The Quick Add field is focused when the app opens. Type a note and press Enter to save it. Use Shift + Enter for a new line.

If you click away from the field, Keeper saves any non-empty text automatically and returns focus to the capture flow.

Notes

Click a note card to open the editor. Notes can have an optional title, Markdown body text, tags, images, links, and checklist-style content.

  • Use the note actions to copy, pin, archive, restore, or delete a note.
  • Emptying a note body and closing the editor deletes that note.
  • Pasted images are stored with the note and inserted as Markdown image references.
  • Markdown checkboxes can be toggled from the rendered preview.
  • Bullet lists continue automatically when you press Enter after a list item.
Links inside rendered note previews open as links, while clicking the rest of the card opens the note.

Views

The sidebar changes which notes are displayed. Each view is its own workspace for a particular kind of cleanup or browsing.

Inbox

Shows active notes, newest first. Pinned notes stay grouped above the rest.

Tag Views

Click any tag in the sidebar to show notes carrying that tag.

Untagged

Shows notes without tags, useful for review sessions and cleanup.

Links

Shows linked notes: anything Keeper detects as containing a URL.

Archive

Shows notes that have been archived out of the Inbox. You can open and restore them.

Trash

Shows deleted notes that can be restored or deleted permanently depending on the action.

Search

Search filters notes by text content. It works across the note body and helps narrow large inboxes quickly, especially when combined with a view-specific cleanup habit.

When a search has no matches, Keeper shows a no-results state instead of the first-run empty state.

Tags

Tags are the main lightweight organization tool. Add tags from the note editor or from the label button on a note card. Selected notes can also be labeled in bulk from the top bar.

  • Tag chips appear on note cards and inside the note editor.
  • Existing tags appear in the sidebar as filters.
  • Double-click a sidebar tag name to rename it across notes.
  • Use the tag icon button in the sidebar to choose a custom icon.
  • Hover a sidebar tag to reveal the delete button for removing that tag.

Bulk Actions

Select notes when you want to label, archive, delete, or export several at once. The top bar shows the count and available actions after selection begins.

  • Use the card selection control, Ctrl/Cmd-click, or long press to toggle selection.
  • Use Shift-click to select a range between the last selected note and the clicked note.
  • Drag across empty grid space to select multiple cards with a selection rectangle.
  • Use Select All or Deselect All for the current displayed set.

Export

Export appears after selecting notes. The export dialog previews exactly what will be copied or downloaded before you commit.

  • Text mode joins selected note bodies with either compact or spaced separators.
  • URLs mode extracts unique links from the selected notes and can sort them alphabetically.
  • Use Copy to clipboard or Download .txt to move the output elsewhere.
  • After a successful copy or download, Keeper offers to permanently delete the exported notes.
The delete-after-export action only appears after export succeeds, so review the preview first and then burn intentionally.

Autotag Rules

Autotag rules apply tags to notes based on URL patterns. Open Settings, choose Autotag Rules, enter a URL regular expression, add one or more tags, and create the rule.

  • Rules are matched case-insensitively, shown as /pattern/i.
  • The tag input suggests existing tags as you type.
  • Rules can be edited or deleted from the rule list.
  • The top-bar autotag button runs rules against notes and reports how many matched.
  • Running rules from the toolbar can archive matching Inbox notes after applying their tags.

Settings

Open Settings from the gear button at the bottom of the sidebar. Settings are grouped into tabs so capture, link handling, backups, and AI features stay separate.

API Key

Configure the OpenRouter key used by the Chat view.

Notes

Control note behavior such as active-tag capture, title badge settings, and tag suggestions.

Link Previews

Enable fetching and displaying previews for notes that contain URLs.

Backup & Import

Download a Keeper archive with SQLite data and optional media, or restore from an archive.

Chat

Chat lets you talk with an AI assistant about your notes. It requires an OpenRouter API key in Settings before use.

  • Choose a model from the model selector.
  • Ask questions in the input bar; press Enter to send or Shift + Enter for a new line.
  • Assistant replies render as Markdown and stream while they are being generated.
  • Potentially destructive actions, such as deleting notes, ask for confirmation.
  • Use the new conversation button to clear the current chat.

Browser Extension

Keeper includes a browser extension for quick web capture. Extension-created notes appear in the app, and Keeper can show an unseen-note count in the tab title when that setting is enabled.

This makes Keeper useful as a web inbox: save links or snippets during browsing, then process them later from the Inbox, Links, Untagged, or tag views.
